\chapter{ИЛЛЮСТРАТИВНЫЙ ЧИСЛЕННЫЙ ЭКСПЕРИМЕНТ}\label{chap4}
Применим изложенный метод для управления системой 
$$x(t+1) = Ax(t) + Bu(t), x(0) = ,$$
$$y(t) = Cx(t) + Du(t), t= 0, \ldots, T$$
размерности $n = 3$ с одним входом и одним выходом,
удовлетворяя ограничения $-1 \leq u(t) \leq 1, \quad -1 \leq y(t) \leq 1, \quad t = \tau, \ldots, T$. Момент начала управления $\tau = n = 3,$ горизонт планирования $T = 11$. 

Сначала были сгенерированы матрицы $A, B, C, D$, априорная траектория $\{u^d, y^d\}$ и начало управляемой траектории $\{u^p(0), u^p(1), u^p(2), y^p(0), y^p(1), y^p(2)\}$  таким образом, что при $\varepsilon = 0.002$ номинальная задача оптимального наблюдения \ref{estimation-lp} решаема. Это даёт нам гарантию того, что для любых векторов ошибок измерения, ограниченных покомпонентно числом $\frac{\varepsilon}{2} = 0.001$ задача \ref{estimation-lp} также будет иметь решение. И действительно, во всех 100 случайным образом сгенерированных векторов ошибок, компоненты которых равномерно распределены в $[-\frac{\varepsilon}{2}, \frac{\varepsilon}{2}]$, как задача оптимального наблюдения так и задача оптимального управления имели решение. А вот уже при ошибках, равномерно распределённых в $[-\varepsilon, \varepsilon]$, из 100 экспериментов 96 были успешны -- что, впрочем, также хороший показатель для случаев,где приведённая оценка не даёт никаких гарантий.
