\chapter*{Заключение}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}


В данной курсовой работе рассмотрена задача оптимального управления линейной стационарной системой в дискретном времени по неточным выходным сигналам. Для исследуемой задачи предложен метод решения, не использующий явной модели системы. Если задача имеет решение при любых реализациях ошибок, ограниченных покомпонентно $\varepsilon$, то предложенный метод гарантированно. Эффективность предложенного метода проиллюстрирована численным примером для трехмерной системы с одним входном и одним выходом.

В дальнейшем интерес представляет получение решения поставленной задачи в случае, когда единственная априорная информация, которой мы располагаем -- одна траектория системы конечной длины -- также известна нам неточно, выходные сигналы в ней содержат ограниченную ошибку.
